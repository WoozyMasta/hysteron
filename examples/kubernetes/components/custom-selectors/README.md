# components/custom-selectors

Demonstration component for multi-instance app-label customization across
selector fields.

This is an example pattern, not a default recommended production component.
Use only when you explicitly need custom app selector propagation in one
namespace.

> [!WARNING]
> Advanced example component.
> Use only when you intentionally run multiple Hysteron instances in one
> namespace and need explicit selector/app-label patching across resources.
